# E-commerce Full Stack Application

This project is a full stack e-commerce application built with .NET (C#) for the backend and React for the frontend.

## Overview

The application provides a REST API for managing products and a shopping cart system, integrated with a React-based user interface. It follows a layered architecture to ensure maintainability and scalability.

## Features

- Product management (CRUD operations)
- Shopping cart functionality
- RESTful API design
- Frontend integration with React
- Data exchange using JSON
- API testing with Swagger and Postman
- SQL database integration

## Technologies

- C# / .NET Framework
- React
- SQL Server
- Swagger
- Postman
- Git

## Architecture

The backend is structured using a layered approach:

- Controllers
- Models
- Services
